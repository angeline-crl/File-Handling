# File Handling
C++ Program that uses Arrays to simulate a database of 'Players'\
Input is done through the Console and Output to Notepad<br/><br/>
**Functions:**
* Store Data in Arrays
* Compute for Average
* Find Maximum and Minimum number in an Array
